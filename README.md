# 🧰 news-app

## expo-cliをインストールする

[Expo — Learn how](https://expo.io/learn)

```bash
$ npm install expo-cli --global
```

## expoプロジェクトを作成する

```bash
expo init my-new-project
cd my-new-project
```

## expoプロジェクトを起動する

```bash
expo start
```
